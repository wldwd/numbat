Numbat Geospatial

Telling stories with data and maps

Numbat Geospatial helps planners, community groups and not-for-profits turn location-based data into maps and insights that inform action.

We specialise in tools that support decision-making, grant applications, and better local outcomes for people walking, riding bikes, using public transport and connecting with nature.

How we can help

  • Geospatial Analysis & Mapping Turn messy or hard-to-use data into clear, useful maps and reports that show patterns, gaps and opportunities.
  • Web Mapping Create interactive maps that support engagement, decision-making and advocacy.
  • Community Atlas A tailored set of reusable maps to support grant applications, project ideas and funding cases.
  • Active Transport Insights Specialist data analysis and visualisation to support walking and cycling initiatives.
